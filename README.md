# Estrutura de Dados em C e Python

Este repositório contém os conteúdos e exercícios da disciplina de **Estrutura de Dados**, com exemplos e códigos em **C** e alguns em **Python**, acompanhando os tópicos ensinados na faculdade mais exercícios extras desenvolvidos por fora.

---

## Estrutura do Projeto

Os arquivos estão organizados para facilitar o seu estudo dos principais conceitos de estrutura de dados:

- **Listas Ligadas / Arrays / Pilhas / Filas /**
  - Cada estrutura clássica está representada por exemplos didáticos em C.
  - Os algoritmos clássicos incluem:
    - Inserção, remoção e busca em listas, pilhas e filas
    - Implementação e travessias de árvores binárias
    - Manipulação de grafos e algoritmos de caminho

- **Exercícios Extras**
  - Scripts com desafios, problemas práticos e resoluções comentadas.

- **Python**
  - Alguns arquivos trazem implementações das mesmas estruturas em Python para comparação.
